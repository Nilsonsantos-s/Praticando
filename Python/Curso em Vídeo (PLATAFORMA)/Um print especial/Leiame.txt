Um print especial:
Faça um programa que tenha uma função chamada escreva(),
que receba um texto qualquer como parâmetro e mostre uma mensagem
com tamanho adaptável.
Ex.:
escreva('Curso de Python no Youtube')
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
        Curso de Python no Youtube
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
-Os caracteres especiais devem acompanhar o tamanho da mensagem.
-A mensagem deve ser sempre centralizada.
